Answer Number 1

  • Web Server:
    Web server adalah perangkat keras atau perangkat lunak yang menyimpan, mengelola, dan mengirimkan konten web kepada pengguna yang mengaksesnya melalui internet. Perangkat lunak web server seperti Apache, Nginx, atau Microsoft IIS umumnya digunakan untuk mengelola permintaan HTTP dari klien (browser pengguna) dan memberikan respons berupa halaman web atau sumber daya lainnya.
  • Domain:
    Domain adalah alamat unik yang digunakan untuk mengidentifikasi lokasi spesifik di internet. Contohnya, "www.namadomain.com." Domain membantu pengguna untuk mengingat dan mengakses website dengan lebih mudah daripada mengingat alamat IP numerik yang terkait dengan server tempat website disimpan. Pendaftaran domain melibatkan proses membayar biaya tahunan kepada registrar domain untuk memastikan kepemilikan unik atas nama domain tersebut.
  • Hosting:
    Hosting adalah layanan yang menyediakan tempat penyimpanan dan sumber daya server untuk menyimpan konten website. Ini dapat berupa hosting bersama (shared hosting), VPS (Virtual Private Server), atau server fisik yang sepenuhnya dikelola. Pemilik website menyimpan file-file website mereka, seperti HTML, CSS, gambar, dan skrip di server hosting. Pengguna kemudian dapat mengakses website tersebut melalui internet.

    • Hubungan antara Web Server, Domain, dan Hosting:

  • Saat seseorang mengetikkan nama domain di browser, permintaan itu dikirimkan ke server DNS (Domain Name System) untuk mengonversi nama domain menjadi alamat IP yang sesuai.
  • Setelah mendapatkan alamat IP, permintaan diteruskan ke server hosting tempat website disimpan.
  • Server hosting menerima permintaan tersebut dan mengirimkan kembali halaman web yang diminta ke browser pengguna melalui web server.
  • Web server mengelola permintaan dan respons, memastikan bahwa konten yang diminta dikirimkan dengan benar ke pengguna melalui internet.
